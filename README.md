##

### O que é o AprendAI?
- É um site para ajudar estudantes de tecnologia com dificuldades no seu processo de aprendizagem;
- Desenvolvido com IA integrada via API da OpenAI, permite o usuário a conversar com um chatbot personalizado;
- Também disponibiliza cursos com certificado para os estudantes;

##

<div align='center'>
<h4>Entendemos que a inteligência artificial poderia ser um tipo de tutor para os estudantes, se usado da maneira correta, e é nisso que nosso projeto é baseado. Uma inteligência artificial que ajude os estudantes de acordo com as suas dificuldades, visando assim um aprendizado direcionado ao que o aluno sente dificuldades em compreender.</h4>
</div>
